# 橘子洲交易体系数据支撑规格书

> **逆向推理说明**：本文档基于橘子洲炒家实战全部直播字幕（约50万字）中提及的所有数据指标和决策依据，逆向推导其交易体系所需的数据支撑。每一个数据项都有字幕原文作为依据。

---

## 一、核心数据架构总览

橘子洲的交易体系是一套**多维信号驱动的决策系统**，所需数据按决策层级分为六大类：

```
                    ┌─────────────────────────┐
                    │    交易决策输出          │
                    │  (买/卖/空仓/仓位)       │
                    └──────────┬──────────────┘
                               │
          ┌────────────────────┼────────────────────┐
          │                    │                    │
     ┌────┴────┐         ┌────┴────┐         ┌────┴────┐
     │ 六维    │         │ 节点    │         │ 窗口    │
     │ 分析    │         │ 识别    │         │ 判断    │
     └────┬────┘         └────┬────┘         └────┬────┘
          │                    │                    │
     ┌────┴────────────────────┴────────────────────┴────┐
     │                基础数据层                          │
     │ 大盘数据 | 情绪数据 | 板块数据 | 个股数据 | 盘口  │
     └───────────────────────────────────────────────────┘
```

## 二、基础数据层（每日必看）

### 2.1 大盘数据

| 数据项 | 用途 | 字幕依据 |
|--------|------|---------|

| 涨跌家数比 | 判断多空力量 | "开盘感觉指数低开，量能萎缩" |
| 指数均线（5/10/20/30日） | 趋势判断 | "沿5日线上行"、"跌破均线" |

### 2.2 投机情绪数据

| 数据项 | 用途 | 字幕依据 |
|--------|------|---------|
| 涨停家数（非ST） | 投机温度 | 字幕中出现 **182次** |
| 跌停家数 | 亏钱效应 | 字幕中出现 **52次** |
| 连板高度（最高板数） | 空间拓展 | "连板高度连续压缩" |
| 连板家数（分层统计） | 梯队完整性 | 出现 **44次** |
| 炸板率 | 分歧程度 | 出现 **56次" |
| 昨日涨停今日溢价 | 赚钱效应 | "昨日涨停今天大面积低开" |
| 晋级率 | 接力意愿 | "晋级率多少" |

### 2.3 板块题材数据

| 数据项 | 用途 | 字幕依据 |
|--------|------|---------|

| 新题材出现 | 切换信号 | "新题材首板" |

### 2.4 竞价数据

| 数据项 | 用途 | 字幕依据 |
|--------|------|---------|
| 昨日涨停股竞价溢价率 | 情绪确认 | 出现 **60次"（竞价类关键词） |

| 竞价封单变化 | 资金态度 | "竞价狂顶" |

## 三、六维分析框架的数据映射

橘子洲的六维框架是**核心决策引擎**，每一维需要的数据如下：

### 维度1：牛熊判断

```
所需数据：
├── 上证指数 日K线（3个月-1年）
├── 创业板指 日K线
├── 两市成交额趋势（日均/周均）
├── 宏观经济数据（政策面）
├── 市场估值水位 (PE/PB)
└── 趋势线/牛熊分界线（年线、半年线）
```

**字幕原文**: "调整当下的市场处在牛市还是熊市，这个很重要。牛市有牛市的玩法，熊市有熊市的玩法。"

### 维度2：多空循环

```
所需数据：
├── 当日涨停家数 vs 跌停家数
├── 涨跌家数比
├── 昨日涨停今日平均溢价
├── 跌停板封单金额（大额封单 vs 撬板）
├── 炸板率
└── 亏钱效应扩散范围（跌停扩散度）
```

**字幕原文**: "跌停板开大概1个多亿，然后翘板了...如果放在过去，跌停板大额封单开盘没有翘板...那么今天不管什么情况，不能去开仓。"

### 维度3：指数状态

```
所需数据：
├── 各指数（上证/深证/创业板）实时点位
├── 指数5/10/20/30日均线
├── 指数分时走势
├── 指数MACD/KDJ等技术指标
└── 指数成交量环比变化
```

### 维度4：题材分析

```
所需数据：
├── 全市场板块指数排行（按涨幅）
├── 各板块涨停家数及占比
├── 板块梯队（龙头几板/跟风几板）
├── 板块总成交额
├── 板块连续上涨天数
├── 题材逻辑强度（政策/产业/事件）
├── 新题材催化事件
└── 板块轮动节奏（龙头板块切换）
```

### 维度5：个股强度

```
所需数据：
├── 个股在板块内的涨停时间（越早越强）
├── 封单额 vs 流通市值比
├── 换手率（10%-30%健康）
├── 个股均线状态（沿5日线上行）
├── 成交额（>10亿有流动性）
├── 历史辨识度（是否老龙头）
├── 龙虎榜席位（游资介入程度）
└── 盘口强弱（扫板/点火/大单）
```

### 维度6：情绪周期

```
所需数据：（综合前五维数据）
├── 综合投机情绪评分
├── 综合市场情绪评分
├── 板块情绪评分
├── 连板高度趋势（突破/压缩）
├── 炸板率趋势（升高/降低）
├── 跌停数趋势（增加/减少）
├── 涨停溢价趋势（正向/负向）
└── 量能趋势（放量/缩量）
```

## 四、节点之法所需数据

节点是橘子洲体系的**核心操作信号**，"节点之法"需要识别以下六种节点：

### 4.1 情绪节点

| 节点类型 | 识别数据 | 触发条件 |
|----------|---------|---------|
| 冰点 | 涨停<15家、连板高度≤2板、跌停>10家开始减少 | 冰点试错 |
| 回暖 | 连板高度突破、涨停连续3日>30家、溢价转正 | 启动仓 |
| 高潮 | 涨停>50家、连板高度≥5板、梯队完整 | 锁仓或分批出 |
| 退潮 | 连板高度压缩、炸板率>50%、跌停激增 | 空仓 |

**字幕依据**: "你看整个市场涨停家数、连板高度、跌停家数这些东西，就大致能判断出情绪处在什么阶段。"

### 4.2 题材节点

| 节点类型 | 识别数据 | 决策 |
|----------|---------|------|
| 启动节点 | 新题材首板出现，有政策催化 | 小仓试错 |
| 确认节点 | 二板确认，板块联动≥3家 | 加仓 |
| 加速节点 | 龙头缩量加速，辨识度极高 | 持有 |
| 分歧节点 | 炸板率飙升，跟风回调 | 去弱留强 |
| 结束节点 | 龙头断板A杀，板块无活口 | 清仓 |

### 4.3 资金节点

```
所需数据：
├── 大单净流入/流出（主力动向）
├── 资金板块流向排行
├── 风格切换信号（情绪→趋势/小盘→大盘）
├── 量能急剧变化（缩量至极/放量突破）
└── 龙虎榜资金动向
```

### 4.4 盘口节点

| 数据项 | 信号含义 |
|--------|---------|
| 竞价量异常放大 | 大资金进场信号 |
| 分时扫板盘口 | 主力强力做多 |
| 炸板后回封时间 | 14:30后回封=最强博弈 |
| 封单变化速度 | 资金态度实况 |
| 均线突破/跌破 | 趋势转折确认 |

## 五、窗口之法所需数据

| 窗口类型 | 数据支撑 | 判断标准 |
|----------|---------|---------|
| 指数窗口 | 指数关键位置（支撑/压力） | 指数在关键均线附近 |
| 题材窗口 | 题材周期（第几天/处于什么阶段） | 题材刚启动未高潮 |
| 情绪窗口 | 情绪阶段判断 | 回暖-高潮期可操作 |
| 时间窗口 | 盘中时间段（早盘/尾盘） | 10:30前最强，尾盘不追 |

**字幕原文**: "窗口其实就是时间与空间的共振点。在确定的时间窗口内集中注意力，在关键节点做极致操作。"

## 六、信号流降噪所需数据（关键过滤机制）

橘子洲区别于其他人的核心是他的**信号过滤系统**——"不寻盘外逻辑，只觅盘中信号"。

### 6.1 需要屏蔽的"噪音"数据

| 噪音类型 | 具体数据 | 屏蔽原因 |
|----------|---------|---------|
| 小道消息 | 各种来源的"内幕" | "不寻盘外逻辑" |
| 社交媒体热议 | 自媒体吹票 | "不带脑子" |
| 他人观点 | 评论区推荐 | "独立思考" |
| 历史路径依赖 | 过去成功经验 | "市场风格会变" |

### 6.2 需要关注的"信号"数据

```
信号优先级（从高到低）：
1. 竞价数据（第一手资金态度）
2. 分时盘口（大单动向/扫板/点火）
3. 板块联动（梯队/跟风/轮动）
4. 量价关系（放量/缩量/堆量）
5. 均线位置（趋势确认/破位）
6. 封单变化（资金决心）
7. 龙虎榜（游资方向确认）
```

**字幕原文**: "我只看盘中信号的，我也不看龙虎榜，我也不听消息，不看基本面。"

## 九、与现有对标数据的对比

| 对标体系 | 相同点 | 不同点 |
|----------|--------|--------|
| **涅槃重升（六大情绪）** | 都用量化情绪指标 | 橘子洲更强调"节点"和"窗口"的时间维 |
| **92科比（情绪周期）** | 情绪周期四阶段类似 | 橘子洲多"道法术器"层次、少"补涨切换" |
| **陈小群（龙头战法）** | 都重视强度和辨识度 | 橘子洲不只看龙头，更看题材和情绪节奏 |
| **涅槃/科比** | 都有盘口和竞价数据 | 橘子洲多"哲学观"处理强弱，降噪体系更完整 |

## 十、最小可行数据产品规格

如果要为橘子洲体系构建数据产品，**最小可行**的数据集为：

### 每日必看数据（10项）

```
1. 上证指数 点位/涨跌幅/成交量
2. 涨停家数 / 跌停家数
3. 连板高度（最高板） / 连板家数
4. 炸板率
5. 昨日涨停今日平均溢价
6. 板块涨幅TOP5 + 各板块涨停家数
7. 板块梯队（1/2/3板各几只）
8. 各板块龙头（涨停时间/封单额/换手率）
9. 竞价溢价率 / 竞价量异动
10. 情绪周期判断（冰点/回暖/高潮/退潮）
```

### 盘中实时数据（5项）

```
1. 涨停/跌停/炸板实时计数
2. 板块资金流向（TOP3板块）
3. 核心个股盘口（封单/换手/分时）
4. 弱转强/分歧转一致信号
5. 尾盘信号（抢筹/砸盘）
```
