## 🧠 已收录游资

### 1. 92科比（Kobe）

> 90后新生代顶级游资，从数万资金起步五年做到数亿。淘股吧第11届百万杯实盘赛半年307%收益获季军。

**Skill**：`kobe-trading` — 情绪周期四阶段 + 龙头·补涨·切换 + 大道无形的全能手法

**四阶段模型**：
| 阶段 | 特征 | 操作 |
|------|------|------|
| 低位震荡 | 题材试错期，主线未明 | 首板/1进2轻仓试错 |
| 主升 | 龙头打高度，梯队完整 | 重仓龙头，分歧买入 |
| 高位震荡 | 龙头分歧加大 | 轻仓防守，不碰中位 |
| 主跌 | 情绪杀跌，亏钱扩散 | 强制空仓，尾盘试错 |

**核心铁律**：
- 要么做高（龙头），要么做低（补涨），不做中位
- 短线本质 = 人气带来的流动性溢价
- 退潮期管住手，宁可错过不可做错

📖 `articles/kobe/`（4篇） · `skills/kobe-trading/SKILL.md`

### 2. 陈小群（Chen Xiaoqun）

> 95后新生代顶流游资，30万起步7年做到10亿，2022年一年32倍封神。常用席位：大连黄河路/大连金马路。

**Skill**：`chen-xiaoqun-trading` — 主线龙头战法 + 弱转强/反核/首阴反包 + 铁血风控 + 八不买六不卖

**情绪四阶段仓位模型**：
| 阶段 | 特征 | 仓位 |
|------|------|------|
| 启动期（冰点） | 连板≤3只，亏钱效应零散 | 10-20%轻仓试错 |
| 发酵期（主升） | 连板10-15家，主线确立 | 50%+重仓锁龙 |
| 高潮一致期 | 连板>15只，缩量加速 | 减至20%，只卖不买 |
| 退潮期 | A杀、天地板频发 | 强制空仓 |

**五大买点**：弱转强接力 ⭐ / 反核低吸 / 首阴反包 / 二板定龙 / 过前高回调

**核心铁律**：只做主线只做总龙 · 买在分歧卖在一致 · 单笔>5%止损 · 有龙做龙无龙空仓

📖 `articles/chen-xiaoqun/`（5篇） · `skills/chen-xiaoqun-trading/SKILL.md`

### 3. 凤凰涅槃·涅槃重升

> 4年100万到1亿封神，淘股吧新时代赵老哥，情绪量化第一人，树干树枝体系创始人。

**Skill**：`fenghuang-niepan-trading` — 六大情绪变量量化 + 树干树枝系统 + 情绪周期五阶段 + 确定性双核（打板+做核心）+ 25条悟道心法

**六大情绪变量**：
| 变量 | 核心指标 |
|------|---------|
| 投机情绪 | 涨停数、连板高度、连板家数、炸板率 |
| 市场情绪 | 指数强度、涨跌比、量能变化 |
| 板块情绪 | 梯队完整度、龙头强度、持续性 |
| 整体市场情绪 | 近期赚钱效应趋势、氛围、消息面 |
| 整体投机情绪 | 高位妖股、涨停溢价趋势、炸板负溢价 |
| 整体板块情绪 | 龙头拉开空间、非一日游、始终活跃 |

**情绪周期五阶段与仓位**：
| 阶段 | 条件 | 仓位 |
|------|------|------|
| 冰点 | 涨停<15、连板≤2、跌停开始减少 | 1-2成试错 |
| 启动/发酵 | 连板突破、涨停>30、梯队完整 | 5-8成 |
| 主升/一致 | 涨停>50、梯队1-2-3板、大盘共振 | 满仓 |
| 分歧 | 涨停骤减但龙头仍封、炸板率40-50% | 3-5成去弱留强 |
| 退潮 | 连板压缩、龙头炸板无回封、跌停激增 | 空仓 |

**两种触发模式**：
- **凤凰涅槃买什么** → 自动采集近3天+今日数据 → 量化六大情绪 → 判断周期阶段 → 筛选标的 → 输出：股票+仓位+买卖点+止损
- **凤凰涅槃复盘** → 同上分析 → 额外输出：操盘反思+纪律检查+经验提炼+最易亏钱6条排查

**25条悟道心法核心**：逆情绪必遭反噬 · 空仓是最高境界 · 情绪冰点买最强波动 · 买在分歧转折点 · 错误交易第二个竞价割

### 4. 橘子洲炒家实战（小橘）

> 信号流交易第一人。基于74篇淘股吧文章+10场直播的系统性提取。不猜、不赌、不意淫——只看市场给出的信号，根据信号做动作。

**Skill**：`juzizhou-trading` — 信号流交易体系 + 六维分析系统 + 8信号复盘框架 + 三法（节点·窗口·盘口）+ 三节奏攻防 + 道法术器心法

| 维度 | 内容 |
|------|------|
| **交易流派** | **信号流**（非情绪流/技术流/基本面流） |
| **核心心法** | 不寻盘外逻辑，只觅盘中信号；于窗口处专注，于节点处极致，于强者处从一而终 |
| **六维分析** | 大盘水位·涨跌家数·量能·昨日涨停溢价·涨停·炸板 |
| **三法** | 节点之法（辨认情绪关键节点）+ 窗口之法（识别交易窗口窗口）+ 盘口之法（读懂买卖盘信号） |
| **三节奏** | 可做节奏（窗口节点共振）+ 难做节奏（窗口节点冲突）+ 不可做节奏（无窗口无节点） |
| **核心铁律** | 信号出现→动作执行；信号没出现→手不动；该弱不弱即为强 |

**两种模式**：
- **橘子洲实时分析** → 自动采集A股数据 → 六维分析 → 节点识别 → 输出决策
- **橘子洲每日复盘** → 按8信号复盘框架输出深度复盘
