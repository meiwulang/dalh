## 🧠 已收录游资

### 1. 92科比（Kobe）

> 90后新生代顶级游资，从数万资金起步五年做到数亿。淘股吧第11届百万杯实盘赛半年307%收益获季军。

**核心体系**：情绪周期四阶段 + 龙头·补涨·切换 + 大道无形的全能手法

**四阶段模型**：
| 阶段 | 特征 | 操作 |
|------|------|------|
| 低位震荡 | 题材试错期，主线未明 | 首板/1进2轻仓试错 |
| 主升 | 龙头打高度，梯队完整 | 重仓龙头，分歧买入 |
| 高位震荡 | 龙头分歧加大 | 轻仓防守，不碰中位 |
| 主跌 | 情绪杀跌，亏钱扩散 | 强制空仓，尾盘试错 |

**核心铁律**：
- 要么做高（龙头），要么做低（补涨），不做中位
- 短线本质 = 人气带来的流动性溢价
- 退潮期管住手，宁可错过不可做错

📖 原始文章见 `articles/kobe/`，技能文件见 `skills/kobe-trading/SKILL.md`

### 2. 陈小群（Chen Xiaoqun）

> 95后新生代顶流游资，30万起步7年做到10亿，2022年一年32倍封神。常用席位：大连黄河路/大连金马路。

**核心体系**：情绪周期四阶段 + 主线龙头战法 + 弱转强/反核/首阴反包 + 铁血风控

**情绪四阶段仓位模型**：
| 阶段 | 特征 | 仓位 |
|------|------|------|
| 启动期（冰点） | 连板≤3只，亏钱效应零散 | 10-20%轻仓试错 |
| 发酵期（主升） | 连板10-15家，主线确立 | 50%+重仓锁龙 |
| 高潮一致期 | 连板>15只，缩量加速 | 减至20%，只卖不买 |
| 退潮期 | A杀、天地板频发 | 强制空仓 |

**五大买点**：弱转强接力 ⭐ / 反核低吸 / 首阴反包 / 二板定龙 / 过前高回调

**核心铁律**：
- 只做主线、只做总龙、拒绝杂毛
- 买在分歧，卖在一致
- 单笔亏损>5%无条件止损
- 有龙做龙，无龙空仓

**经典战役**：浙江建投（单票5000万+）/ 中交地产反核（单票破亿）

📖 原始文章见 `articles/chen-xiaoqun/`，技能文件见 `skills/chen-xiaoqun-trading/SKILL.md`
