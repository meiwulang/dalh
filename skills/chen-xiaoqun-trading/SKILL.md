## 一、核心交易哲学

### 顶层三句话

> **只做主线、只做总龙、拒绝杂毛**

1. **主线定义**：政策、产业催化下资金集体扎堆的唯一主流题材，支线、冷门小众题材一律放弃
2. **总龙筛选标准**：
   - 能带动板块至少5只个股跟风涨停
   - 题材逻辑硬核（政策/产业/事件驱动）
   - 流通盘适中（50–200亿）
   - 辨识度拉满
   - 独立庄股、过气老龙头绝不参与
3. **核心思路**：赚市场情绪合力的钱，不主观预判涨跌，全程跟随资金动向；**宁可空仓错过机会，绝不逆势乱做被套**

### 核心世界观

- **周期大于图形，题材逻辑大于技术形态**
- **小资金做大靠赔率，靠龙头主升大利润覆盖多次小止损**
- **围著主线做，多研究龙头，少看杂毛**
- **买在分歧，卖在一致**
- **有龙做龙，无龙空仓，宁可错过，绝不做错**

## 三、四大招牌实战买点手法

### 1. 弱转强接力 ⭐（最擅长、最核心）

| 维度 | 内容 |
|------|------|
| 定义 | 前一日烂板/炸板/低开走弱 → 次日竞价转强 |
| 判断标准 | 竞价温和高开（2%-5%）、分时承接有力、放量不跳水 |
| 买入方式 | 半路或打板介入 |
| 核心逻辑 | 分歧越大，后续溢价越高；分歧转一致是确定性最高的买点 |
| 经典案例 | 中交地产跌停后次日弱转强，三日获利超40% |

**弱转强4次节点（贯穿龙头全生命周期）**：
| 次数 | 出现阶段 | 对应情绪周期 | 特征 |
|------|----------|-------------|------|
| 第1次 | 2-4板 | 启动期 | 分歧转一致，首板后的首次弱转强 |
| 第2次 | 4-7板 | 发酵期 | 缩量洗盘结束，市场合力共识，开启连板加速 |
| 第3次 | 7板以上 | 高潮期 | 高位洗盘再封板，极致博弈 |
| 第4次 | 见顶后 | 退潮期余温 | 龙回头，反弹出货 |

### 2. 龙头反核低吸（胆大心细）

| 维度 | 内容 |
|------|------|
| 定义 | 总龙因大盘/监管/情绪被动跌停，但题材逻辑未破，尾盘低吸 |
| 前提条件 | 仅限市场总龙；题材产业逻辑未破；板块未集体崩盘 |
| 买入方式 | 尾盘跌停板低吸撬板 |
| 风控 | 跟风杂毛绝不撬板；次日若不涨停全部卖出 |
| 经典案例 | 2022年中交地产跌停单日买入8604万，次日弱转强连板，21天11板，单票盈利破亿 |

### 3. 首阴反包战法

| 维度 | 内容 |
|------|------|
| 定义 | 主线总龙第一次收阴线调整，次日博弈反包 |
| 买入条件 | 次日放量高开、弱转强信号确认 |
| 放弃条件 | 首阴放量崩盘无承接 → 直接放弃 |
| 操作 | 跟进博弈二波行情 |
| 案例 | 浙江建投两轮主升全程运用该手法 |

### 4. 二板定龙打板

| 维度 | 内容 |
|------|------|
| 首板 | 小仓试错，观察板块效应和题材强度 |
| 二板确认 | **竞价高开≥3%、10:30前封板、成交量>首板1.2倍、能带动板块涨停** |
| 操作 | 确认板块合力、主线地位后打板重仓 |
| 核心规则 | **只打主线龙头板，后排跟风板一概不参与** |
| 淘汰标的 | 首板缩量加速无换手的票、无板块效应的独立首板 |

**梯度建仓体系（来自实战总结）**：
- 量价/筹码/均线**三维全达标** → 6成仓
- 量价+筹码**二维达标** → 3成仓
- **仅量价达标** → 1成仓试错
- 买入后2个交易日未达预期 → 果断离场

### 5. 过前高回调形态（小资金专用）⭐

> 陈小群原话：「牛市里，贪心是亏损的根源，专注才是盈利的密码。」早年他本金30W内盲目广撒网同时持8只票，1个月缩水至12W，痛定思痛后专攻此形态，牛市中实现翻倍。

| 维度 | 内容 |
|------|------|
| 适用人群 | 本金30万以内的小资金散户 |
| 形态定义 | 股价突破前高后缩量回调，回踩前期高点支撑位不破 |
| 核心原理 | 突破前高位置是多空对决核心区域，形成密集筹码区，即主力成本区；主力不会让股价轻易跌破自己的成本区，回调至此会主动承接形成强支撑 |
| 核心逻辑 | 专注单一形态反复做精，避免小资金「广撒网」分散精力 |

**量价确认标准**：
- **突破时**：成交量较前期均量放大30%以上，且连续2个交易日保持放量（真突破，非诱多）
- **回调时**：成交量缩量，缩量幅度不低于前期放量峰值的50%（抛压极小，主力未出货）
- **涨时放量、跌时缩量**：最健康的量价配合形态
- ⚠️ 若回调时**放量** → 主力出货非洗盘，果断规避

**支撑确认标准**：
- 优先参考**30日均线**（中期平均持仓成本，支撑力度最强）
- 可搭配**10日均线**形成短期+中期双重支撑
- 股价回调至均线附近 + 前高支撑位 → **共振支撑**，进场确定性最高

**操作节奏**：
1. 突破前高放量确认 → 不急于进场，等回调
2. 回调缩量企稳，回踩前高/均线支撑位不破 → 顺势布局
3. 二次放量启动时 → 加仓
4. 回调幅度通常不超过前期上涨幅度的**15%**，过大则支撑失效

**变体：横盘代替回调**：
- 突破前高后不回调，陷入窄幅横盘震荡
- 横盘期间成交量较突破时缩减60%以上（主力清理浮筹）
- 横盘周期通常5-8个交易日，震荡幅度≤3%，K线小阴小阳
- 横盘结束后启动拉升 → 主升浪起点进场

**历史验证**：陈小群早年30万亏至12万后专攻此形态，牛市中实现翻倍

## 六、牛市稳赚铁律：进有八不买、出有六不卖

> 陈小群原话：「牛市一来，多数人都觉得闭眼买都能赚，结果往往是牛市赚的小钱，一波回调就全吐回去，甚至倒亏本金。牛市不是靠运气赚钱，而是靠纪律守住利润。」
> 
> 牛市的核心是资金扎堆、情绪狂热，主力会利用散户的贪婪制造虚假上涨信号，诱导高位接盘。没有清晰的买卖边界，看到涨就追、看到跌就卖，完全被情绪牵着走，就成了主力出货的接盘侠。

### 🔴 进有八不买（买入前逐条检查）

| 序号 | 铁律 | 判断标准 | 市场逻辑 |
|------|------|---------|---------|
| 1 | **短期暴涨超100%、无业绩支撑的纯题材股不买** | 1-2个月涨幅超100%，扣非净利润连续亏损，主营业务与题材关联度低 | 上涨全靠情绪，无业绩兜底，资金撤离后断崖式下跌 |
| 2 | **高位放量滞涨的票不买** | 股价连续大涨到高位，成交量大幅放大但股价不涨反震荡 | 主力疯狂卖、散户疯狂买，多空力量彻底反转，后续大概率开启下跌 |
| 3 | **尾盘15分钟急拉、无量配合的票不买** | 尾盘最后15-30分钟股价突然直线拉涨，但成交量没同步放大 | 主力用少量资金拉高收盘价美化K线，制造强势假象，吸引散户次日追高好出货 |
| 4 | **基本面有重大瑕疵的票不买** | 财务造假、重大诉讼、大股东大幅减持、主营业务停滞、被监管处罚 | 牛市炒作会掩盖风险，但行情稍有风吹草动，风险集中爆发，暴跌速度远超其他票 |
| 5 | **连续一字板、开板后巨量换手的票不买** | 连续一字涨停后开板，成交量瞬间放大、换手率远超平时 | 一字板期间筹码没充分交换，主力获利极丰厚，开板巨量换手即主力集中出货 |
| 6 | **股价严重偏离5/20日均线超20%的票不买** | 股价连续大涨，远离5/20日均线超过20%，属于严重超买 | 没有均线支撑，随时会回调修复向均线靠拢，相当于买在短期高点 |
| 7 | **大除权后1个月内放量的票不买** | 高送转/大比例分红除权后1个月内，成交量放量上涨 | 主力利用除权后"低价"假象吸引散户买入趁机出货，后续大多持续阴跌 |
| 8 | **行业夕阳、无发展前景的票不买** | 所在行业市场需求萎缩、无技术创新、政策不支持 | 上涨只是短暂跟风无持续性，涨幅远跑输主线行业，浪费资金和时间 |

### 🟢 出有六不卖（持有中逐条检查）

| 序号 | 铁律 | 判断标准 | 市场逻辑 |
|------|------|---------|---------|
| 1 | **主线板块龙头、趋势完好不卖** | 板块内最先涨停、涨幅最大、带动板块效应最强；沿5/20日均线稳步上涨，未出现高位放量出货信号 | 牛市主线龙头的主升浪往往能带来翻倍收益，别因小幅震荡提前卖出错过大头利润 |
| 2 | **低位放量启动、均线多头排列不卖** | 股价长期低位横盘后成交量持续放大突破横盘区间，5/10/20日均线形成多头排列 | 主力建仓完毕即将拉升的明确信号，刚启动上涨空间完全没打开 |
| 3 | **上升趋势、高点和低点逐步抬高不卖** | 高点不断创新高、低点不断抬高，每次回调不跌破前期低点 | 主力控盘力度强，回调只是正常洗盘，洗掉不坚定散户为后续拉升减轻压力 |
| 4 | **缩量回调、不破20/30日均线不卖** | 回调时成交量明显萎缩，股价不跌破20日或30日均线 | 缩量说明没有大资金出逃，只是散户恐慌卖出，主力仍在控盘 |
| 5 | **MACD水上金叉、红柱持续放大不卖** | MACD在零轴上方形成金叉，且红柱持续放大 | 多头力量占绝对优势，股价上涨动力充足，会继续保持上升趋势 |
| 6 | **早盘急跌无放量、午后快速回升不卖** | 早盘急跌时成交量没放大，午后股价大概率快速回升 | 没有主力资金出逃，纯属市场情绪引发的恐慌砸盘（洗盘行为） |

### 实操落地指南

1. **选股时**：先逐一对照「八不买」，只要占一条直接排除，不做任何纠结
2. **持股时**：对照「六不卖」，只要符合一条就坚定持有，不被盘中短期波动影响
3. **卖出时**：只看信号不看情绪。触发卖出信号（跌破均线、高位放量出货、趋势破坏等）果断卖出；没触发信号耐心持股，不提前止盈
4. **仓位管理**：主线龙头配置5-6成，低位优质票2-3成，留2成机动资金应对回调

## 八、7天交易核心原则（实战口诀）

| 序号 | 原则 | 核心内容 |
|------|------|---------|
| 1 | **知题材、明逻辑** | 买票前一定要清楚题材预期和公司基本面。题材预期决定行情大小，基本面决定逻辑是否正宗 |
| 2 | **早盘急跌不卖票** | 早盘急跌不要恐慌卖出，等企稳后看强度再决定，可避免情绪抛盘，有机会收大长腿 |
| 3 | **尾盘大涨减仓，尾盘大跌次日进** | 尾盘拉升常为游资刻意制造的抢筹假象，为次日出货；尾盘大跌次日通常有惯性下跌，低点进场盈亏比高 |
| 4 | **10:30前涨停最强** | 越早涨停越强，10:30前涨停+大单封板+全天不开板=强势板；10:30后多为跟风；尾盘涨停为投机板不碰 |
| 5 | **短线看五日线** | 五日线是短线攻击线，沿五日线上行为强势品种放心持有；跌破五日线则短期转弱，分批离场 |
| 6 | **弱水三千，只取一瓢** | 先做加法（不断学习各种方法），再做减法（只做让自己赚钱的方法），建立属于自己的交易模式 |
| 7 | **适当空仓是盈利第一步** | 连续亏损后必须强制空仓，调整节奏再出手；心态徘徊在悟道与销户之间时要停下来 |

## 十、完整操作流程（每日）

### 09:15-09:25 竞价阶段
- 看总龙头竞价是否超预期/不及预期
- 弱转强信号：竞价温和高开2%-5%，放量不跳水
- 二板定龙条件：**竞价高开≥3%、成交量>首板1.2倍**
- 若龙头竞价核按钮 → 警惕退潮

### 09:30-10:00 开盘确认
- 看昨日涨停票的溢价情况
- 看龙头股开盘走势（分歧低开但承接强 → 可介入）
- 10:30前封板的票重点关注（越早涨停越强）

### 10:00-11:30 盘中观察
- 龙头股缩量板谨慎接力
- 板块联动效应判断主线强度
- 分歧转一致的信号识别
- 关注跟风股是否批量炸板（降仓避险信号）

### 13:00-15:00 尾盘决策
- 尾盘拉升 → 减仓（诱多假象）
- 尾盘大跌 → 准备次日低吸
- 确认次日操作计划
- 尾盘15分钟无量急拉 → 不追

### 本周可执行计划（来自陈小群实战总结）

1. **每日盘前定主线**：看涨停家数排名、板块成交额排名、政策/事件驱动
2. **选龙头**：主线内选板块带动≥5家涨停、流通盘适中、辨识度最高的票
3. **等拐点**：二板定龙 / 弱转强 / 反核信号出现，1成仓试错
4. **破5%止损或2日未达预期** → 果断离场
5. **发酵期加到5成**，高潮期分批减，退潮期空仓

### 复盘要点
- 检查情绪周期所处阶段
- 确认主线是否持续
- 评估龙头强弱状态
- 规划次日买卖策略
- 检查是否违反「八不买六不卖」纪律
