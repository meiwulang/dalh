## 阶段一：数据采集（所有分析的前置条件）
## ═══════════════════════════════════════

> **原则**：先抓数据，再谈分析。不采集数据不能输出结论。
> **接口**：使用已有 `stock-data-fetcher` 技能的全部接口。

### 第一步：获取当前日期时间

```bash
date "+%Y-%m-%d %H:%M %A"
```

### 第二步：采集大盘数据（新浪指数快照）

```bash
curl -s "https://hq.sinajs.cn/list=s_sh000001,s_sz399001,s_sz399006,s_sh000688,s_sh000300,s_sh000852" \

**分析公式**：
```
多空强弱比 = 涨停家数 / (跌停家数 + 1)

## 阶段三：节点识别（信号降噪过滤后）
## ═══════════════════════════════════════

> **核心**：在六维分析的基础上，过滤噪音信号，识别真正的交易节点。

### 3.1 信号降噪过滤

根据橘子洲"不寻盘外逻辑，只觅盘中信号"原则，过滤掉以下噪音：

| 过滤类型 | 屏蔽内容 | 保留内容 |
|----------|---------|---------|
| 消息噪音 | 各类传闻、自媒体吹票 | 政策文件、官方数据 |
| 情绪噪音 | 个人恐惧/贪婪 | 客观情绪数据 |
| 他人噪音 | 评论区观点、大V推荐 | 盘面语言、资金动向 |
| 历史噪音 | 过去成功路径依赖 | 当前数据、实时信号 |

### 3.2 六类节点识别

#### ① 情绪节点

| 节点类型 | 触发条件 | 操作指引 |
|----------|---------|---------|
| **冰点节点** | 涨停<15家 + 连板高度≤2板 + 跌停>10家减少 + 溢价由负转正 | ✅ **试错窗口**：首板/1进2轻仓试错 |
| **回暖节点** | 连板高度突破 + 涨停连续>30家 + 溢价正向 | ✅ **加仓窗口**：确认方向，加仓核心 |
| **高潮节点** | 涨停>50家 + 高度≥5板 + 梯队完整 | ⚠️ **持有/减仓**：缩量加速则分批出 |
| **退潮节点** | 连板高度压缩 + 炸板率>50%持续 + 跌停激增 | ❌ **空仓窗口**：强制空仓，不操作 |

#### ② 题材节点

| 节点类型 | 触发条件 | 操作指引 |
|----------|---------|---------|
| **启动节点** | 新题材首板出现 + 政策/事件催化 | 小仓关注 |
| **确认节点** | 二板确认 + 板块联动≥3家 | 加仓 |
| **加速节点** | 龙头缩量加速 + 辨识度极高 | 持有不动 |
| **分歧节点** | 炸板率飙升 + 跟风回调 | 去弱留强 |
| **结束节点** | 龙头断板A杀 + 板块无活口 | 清仓不回头 |

#### ③ 量价节点

| 信号 | 识别数据 | 含义 |
|------|---------|------|
| **放量突破** | 成交量>5日均量1.5倍 + 价格突破关键位 | 新趋势启动 |
| **缩量回调** | 回调时成交量<放量期50% | 健康洗盘 |
| **地量冰点** | 成交额缩至近期最低 | 拐点临近 |
| **放量滞涨** | 量放大但价格不动 | 主力出货 |

#### ④ 盘口节点（橘子洲核心）

| 信号 | 识别数据 | 含义 |
|------|---------|------|
| **竞价量异动** | 竞价量>昨日总成交量10% | 大资金进场 |
| **扫板盘口** | 大单连续扫货冲击涨停 | 主力强力做多 |
| **14:30后回封** | 尾盘炸板后强势回封 | 最高博弈价值 |
| **封单变化** | 封单急剧减少/增加 | 资金态度转变 |
| **破均线** | 分时跌破均价线不回 | 短期转弱信号 |

#### ⑤ 窗口节点

| 窗口类型 | 判断标准 | 操作意义 |
|----------|---------|---------|
| **时间窗口** | 10:30前涨停的票 | 最强，重点关注 |
| **空间窗口** | 股价在关键均线/支撑位附近 | 盈亏比合适的入场点 |
| **情绪窗口** | 情绪回暖→高潮阶段 | 可操作期 |
| **题材窗口** | 题材刚启动，未高潮 | 还有空间 |

#### ⑥ 强弱路径节点（橘子洲独创哲学观）

基于橘子洲的"强弱路径体系"，识别四种路径：

```
加速 → 分歧 → 再加速 → 再分歧 → 结束
  ↑        ↓
  └── 一致 ──┘
```

| 路径位置 | 识别特征 | 操作 |
|----------|---------|------|
| 第一次加速 | 首次放量突破，板块共振 | 跟随 |
| 第一次分歧 | 炸板、放量震荡，但不破关键位 | 低吸 |
| 再加速 | 缩量加速，辨识度极高 | 持有 |
| 再分歧 | 巨量震荡，跟风退潮 | 减仓 |
| 结束 | A杀、跌停、板块团灭 | 清仓 |
