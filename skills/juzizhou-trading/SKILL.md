## 第一篇·交易哲学：底层认知框架
## ════════════════════════════════════════════════════════

### 一、交易的本质

> **"交易的本质不是预测，是应对。"**
> **"市场不是猜出来的，是走出来的。"**

小橘反复强调：预测是散户最大的误区。"明天是涨是跌"这个问题本身就错了。正确的问题是："如果涨，我做什么？如果跌，我做什么？"

他的体系中没有任何预测成分。只有：
- **信号出现** → **动作执行**
- 信号没出现 → 手不动

### 二、信号流 vs 其他流派

| 流派 | 依据 | 弱点 |
|------|------|------|
| 情绪周期流 | 冰点→启动→高潮→退潮 | 周期判断主观，容易数错 |
| 龙虎榜流 | 游资席位动向 | 看到的已经是二手信息 |
| 技术分析流 | MACD/KDJ/均线 | 滞后，容易陷入指标陷阱 |
| 基本面流 | PE/业绩/研报 | 短线来不及反应 |
| **信号流（小橘）** | **盘面客观信号组合** | **需要纪律和耐心** |

信号流的核心：**不猜、不赌、不意淫**。只看市场已经给出的信号，根据信号做动作。

### 三、三句话揭开交易真相

**第一句：该弱不弱，视为强；该强不强，视为弱。**
这是小橘体系的核心判断准则。用在三个层面：

| 层面 | 该弱不弱（=强） | 该强不强（=弱） |
|------|----------------|----------------|
| 指数 | 利空不跌 | 利好不涨 |
| 板块 | 大盘跌它横盘 | 大盘涨它不跟 |
| 个股 | 炸板后有人接 | 该封板却烂板 |

**第二句：指数是背景，个股是战场。**
指数决定仓位方向，但真正扣扳机的依据是个股自身的信号。

**第三句：分散押注的市场，是主线还没有完全确立的市场。**
资金没有形成共识时，接力要更谨慎。

### 四、体系架构：道→法→术→器→进阶→知行合一→哲学

```
道  →  法  →  术  →  器  →  进阶篇  →  知行合一  →  哲学
↓       ↓       ↓       ↓         ↓            ↓           ↓
心性    节点    竞价    工具     穿越牛熊     管住手     强弱观
修养    盘口    技法    使用     各市场玩法   训练       哲学升华
        窗口
```

- **道**：心性修养——根基，先讲道再讲法
- **法**：盘口之法、节点之法、窗口之法
- **术**：竞价之术、进价之术
- **器**：工具的使用
- **进阶篇**：穿越牛熊（牛市/熊市/僵尸市/量化）
- **知行合一**：管住手训练
- **哲学**：用哲学观处理强弱好坏

> **核心观点**：没有到"道"的层面，术掌握得越多，实际上亏得越快。

## 第二篇·8信号复盘框架（完整版）
## ════════════════════════════════════════════════════════

这是小橘每篇复盘的骨架，8个信号按顺序逐一分析：

### 信号一：K线信号（日线形态）

**观察内容：**
- 各大指数（上证/深证/创业板/科创50）的日K线形态
- 阳线/阴线/十字星/长上影/长下影/光脚阳线
- 是否突破关键均线（5日/20日/60日）
- 组合形态（底分型/顶分型/孕线/吞没）

**判断规则：**
- 连续阳线+放量 = 进攻信号
- 十字星+缩量 = 分歧信号
- 长上影+放量 = 抛压信号
- 光脚阴线 = 空头强势

> "昨天增量跳空长阳，今天缩量十字星，两个字：分歧。"

### 信号二：量能信号（成交额）

**观察内容：**
- 两市总成交额（万亿级别）及环比增减
- 放量位置（高位放量/低位放量/突破放量）
- 量价关系是否健康

| 场景 | 含义 |
|------|------|
| 低位放量上涨 | 资金进场，健康 |
| 高位放量滞涨 | 换手加大，见顶信号 |
| 缩量上涨 | 一致性过高，警惕反杀 |
| 缩量下跌 | 惜售，可能有反弹 |
| 放量下跌 | 止损盘涌出，警惕 |

**特殊技巧：** 对比上午半日成交和昨日全天成交的比例。如果上午半日已经达到昨日全天的60%以上，今日大概率放量；如果不到40%，大概率缩量。

### 信号三：强趋信号（强势股状态）

**观察内容：**
- 当前市场最高连板标的及状态（加速/分歧/断板）
- 断板高标的负反馈程度（跌停/A杀/温和断板）
- 一字板数量和质量（题材驱动还是公告驱动）
- 涨停梯队完整性（龙头+中军+跟风是否齐全）

**小橘量化标准：**
- 一字板数量 > 10个 = 进攻期
- 一字板 5-8个 = 防守期
- 一字板 < 5个 = 等待期

**关键观察：** 一字板的质量比数量更重要；龙头断板的方式（温和断板≠ A杀）；连续三天缩量滞涨=进攻乏力信号。

### 信号四：指数信号（多指数对比）

**观察内容：**
- 上证、深证、创业板、科创50、沪深300、北证50
- 谁领涨？谁领跌？谁在护盘？
- 是否存在剪刀差（沪强深弱 / 沪弱深强）
- 涨跌家数比

### 信号五：题材涨幅信号

**观察内容：**
- 当日涨幅前三的题材/板块及上涨逻辑
- 梯队的完整性（龙头+助攻+跟风）
- 是"新方向"还是"老方向反弹"？

**区分方法：** 新方向=首次出现+逻辑新鲜+低位启动+梯队完整；老方向反弹=旧闻重提+只有龙头+高位反抽。

> "前排涨停的时候，后排能跟上，这才是梯队。后排跟不上的，叫孤军深入。"

### 信号六：题材跌幅信号

**观察内容：**
- 当日跌幅前三的题材/板块及下跌原因
- 是"获利了结"还是"逻辑破坏"？
- 跌幅榜中是否存在"被错杀"的机会

**错杀识别规则：** 被动下跌（被大盘带下，非内部利空）+ 核心股抗跌（龙头跌幅<板块平均）+ 缩量下跌。

> "你在跌幅榜里看到的是风险，我在跌幅榜里看到的是被错杀的猎物。"

### 信号七：节点信号（周期定位）

#### 六阶段周期模型

```
异动期 → 加速期 → 分歧期 → 转一致 → 高潮期 → 退潮期
```

| 阶段 | 策略 | 仓位参考 |
|------|------|---------|
| **① 异动期** | 关注，小仓试错 | 1-2成 |
| **② 加速期** | 可参与前排核心 | 3-5成 |
| **③ 分歧期** | 减仓或空仓观望 | 0-3成 |
| **④ 转一致** | ⭐ **最佳买点，打最强** | 3-7成 |
| **⑤ 高潮期** | 持股或逐步兑现 | 3-5成 |
| **⑥ 退潮期** | **空仓，不出手** | 0成 |

#### 如何判断分歧转一致（三个条件）
1. **竞价阶段：** 前一天分歧的核心标的，今天竞价从深水拉到平盘或红开（弱转强）
2. **开盘5分钟：** 开盘后快速拉升，分时均线向上，不破开盘价
3. **板块响应：** 同板块有助攻票跟随，不是单打独斗

### 信号八：强度信号（综合评估）

**评估维度（5星制）：** 指数强度、主线强度、连板情绪、赚钱效应、量能支撑

| 评分 | 窗口状态 | 策略 |
|------|---------|------|
| **窗口开启**（4-5星） | 可以做，积极参与 | 5-7成仓位 |
| **混合信号**（2.5-4星） | 谨慎，提高标准 | 2-4成仓位 |
| **窗口关闭**（1-2.5星） | 不做，空仓等待 | 0成仓位 |

## 第三篇·三节奏攻防体系（操作节奏）
## ════════════════════════════════════════════════════════

### 三大节奏定位

| 节奏 | 触发条件 | 仓位 | 动作 |
|------|---------|------|------|
| **进攻节奏** | 窗口开启+主线清晰+高标健康+梯队完整 | 5-7成 | 推重仓，打核心 |
| **防守节奏** | 窗口模糊+主线不明+但有局部活口 | 2-4成 | 轻仓试探，快进快出 |
| **等待节奏** | 窗口关闭+老方向出清+新方向未现 | 空仓 | 只看不动，全逆回购 |

**自动切换规则：**
```
进攻节奏 = 强趋全线红开 + 一字板＞10个 + 老核心企稳 + 新方向批量涨停
防守节奏 = 强趋涨跌互现 + 一字板5-8个 + 老方向出清中 + 新方向试探中
等待节奏 = 强趋全线绿开 + 一字板＜5个 + 老核心跌停 + 新方向一日游
```

> "这三种节奏的切换，不是小橘主观判断的，而是由信号组合自动触发的。没有模糊地带，没有主观猜测。"

## 第四篇·选股与买点体系
## ════════════════════════════════════════════════════════

### 一、集中选股法："瞄准镜"理论

**第一步：** 收盘后，从涨停板里找出最强板块
**第二步：** 从该板块中选出最强的1-2只个股（每天关注池不超过10只）
**第三步：** 次日只盯着这10只票，出现买点的扣扳机，没有就空仓

> "关掉杂音，删掉杂毛。把你的瞄准镜，只对准最强的那个。"

### 二、买点体系：三个可靠买点

| 买点 | 场景 | 仓位 | 胜率 |
|------|------|------|------|
| **① 首板**（第一次节点） | 方向异动期的第一只涨停 | 1-2成 | 盈亏比好，胜率一般 |
| **② 分歧转一致**（第二次节点）⭐ | 方向经历分歧后资金重新确认 | 3-7成 | 最佳，胜率高+盈亏比好 |
| **③ 加速后再分歧**（第三次节点） | 主线走2-3天后的第二次分歧 | 1-2成 | 一般，仅核心票值得博弈 |

### 三、第十讲核心补充：接力的11种买点（第一组）

*来源：2026.06.26直播第十讲【法篇】买点之法*

**第一组：预期差买点（底层逻辑—博弈认知差）**

| 手法 | 核心逻辑 | 操作要点 |
|------|---------|---------|
| **① 超预期** | 该弱不弱是为强 | 前一天分歧烂板，次日竞价从深水拉到红开 |
| **② 弱转强** | 弱势转强势的信号确认 | 分时从水下翻红，量能配合 |
| **③ 错杀** | 被动下跌中的错杀机会 | 被大盘带下来的优质标的 |

> "所有买点里面，最底层逻辑的是预期差的买点。手法不同，但底层逻辑都是博弈的认知差。"

### 四、左峰突破法

**放量突破左峰** → 压力变支撑，可以买入
**缩量触碰左峰** → 大概率遇阻回落，等待
**触碰后回踩不破** → 确认支撑，加仓机会

## 第五篇·卖点与风控体系
## ════════════════════════════════════════════════════════

### 一、卖点体系：5种离场信号

| 信号 | 动作 |
|------|------|
| **预期内的走强** | 持有 |
| **预期外的走弱** | 无条件走人 |
| **该强不强** | 兑现，不格局 |
| **该弱不弱** | 可以多拿一下 |
| **放量滞涨** | 卖出，见顶信号 |

### 二、止损体系

- 固定止损：买入价的-5% → 无条件止损
- 动态止损：浮盈超过10%后，回撤到成本价 → 至少保本出
- 时间止损：买入后3天未按预期走 → 出局

### 三、仓位管理

| 市场状态 | 总仓位 | 单票上限 |
|---------|--------|---------|
| 窗口开启 | 5-7成 | 3成 |
| 防守期 | 2-4成 | 1.5成 |
| 窗口关闭 | 空仓 | 0 |
| 连续亏损后 | 强制空仓 | — |

### 四、风控铁律

1. 单票最大亏损不超过总资金的3%
2. 连续亏损两笔 → 强制休息1-3天
3. 龙头断板日：不接力，不抄底
4. 退潮期：每一次反弹都是诱多，不参与
5. 窗口关闭：空仓也是策略
6. 最难时，不做最急的决定

> "退潮期的每一次反弹，都是诱多。抄底不是技术，是赌博。"

## 第六篇·心理与修炼体系
## ════════════════════════════════════════════════════════

### 一、空仓哲学

空仓不是"没办法所以空仓"，而是"空仓也是进攻的一部分"。

**空仓的三大作用：** 守住本金、保持清醒（持仓时判断力下降40%）、等待机会来了才有子弹。

**如何克服空仓焦虑：** 写复盘笔记、做历史回测、建立"前置认知"。

> "以前空仓的时候浑身难受，现在空仓的时候浑身自在。别人把空仓当煎熬，你把空仓当修炼。"

### 二、管住手 = 建立边界

> "回撤控制不好的原因，就是你没有边界。什么叫边界？就像你去上班，8点半必须到办公室打卡，这就是边界。"

**边界就是在交易中明确规则：** 什么情况下可以买？什么情况下必须卖？什么情况下不能操作？单笔最大亏损多少？

### 三、交易境界的七个层次

```
追涨杀跌 → 学习模式 → 打造模式 → 模式成熟 → 管住手 → 如如不动 → 无限境界
```

> "交易最大的收获不是钱，是变得无敌。什么叫无敌？就是生老病死，所有的东西在你的世界里面，变得不重要。"

## 第七篇·数据驱动的六维分析系统（可执行）

本系统将上述理论框架转化为可执行的数据分析流程，每次运行分为四个阶段：

```
阶段一：数据采集  →  阶段二：六维分析  →  阶段三：节点识别  →  阶段四：输出决策
                                       ↓
```

###  阶段一：数据采集（所有分析的前置条件）

> **原则**：先抓数据，再谈分析。不采集数据不能输出结论。
> **依赖**：使用 `stock-data-fetcher` 技能的全部接口。

**第一步：获取当前日期时间**
```bash
date "+%Y-%m-%d %H:%M %A"
```

**第二步：采集大盘数据（新浪指数快照）**
```bash
curl -s "https://hq.sinajs.cn/list=s_sh000001,s_sz399001,s_sz399006,s_sh000688,s_sh000300,s_sh000852" \
```

###  阶段二：六维分析（框架研判）

> **核心原则**：六个维度逐一分析，缺一不可。单维度不决策。

#### 维度1：牛熊判断
**分析依据**：上证指数/创业板指近3个月走势、两市成交额趋势、均线排列（5/10/20/30/60日）

#### 维度2：多空循环
**分析公式**：
```
多空强弱比 = 涨停家数 / (跌停家数 + 1)
赚钱效应 = 昨日涨停今日平均溢价
```

#### 维度3：指数状态
**分析要点**：各指数是否处于关键均线位置、指数间是否共振/背离、分时黄白线关系

#### 维度4：题材分析
**分析输出**：板块涨幅TOP10 + 涨停数 + 梯队结构 + 主线判断（最强主线/支线轮动/新题材切换）

#### 维度5：个股强度
**分析框架**（橘子洲7类强弱法）：
```
小弱 → 小强 → 中强 → 大强 → 巨强 → 反量化 → 前驱
(极弱)  (一般)  (中等)  (强)   (极强)  (做空)  (领涨)
```

#### 维度6：情绪周期
**综合前五维数据判断**：冰点/启动/主升/分歧/退潮

| 数据维度 | 冰点 | 启动/发酵 | 主升/一致 | 分歧 | 退潮 |
|---------|------|----------|----------|------|------|
| 涨停数 | <15 | 30-50 | >50 | 骤减 | <20 |
| 连板高度 | ≤2板 | 突破近期 | ≥5板 | 压缩 | 压缩 |
| 炸板率 | — | <30% | <30% | 40-50% | >50% |
| 跌停数 | >10减少 | <5 | <3 | 增加 | 激增 |
| 量能 | 地量 | 放量 | 共振放量 | 缩量 | 缩量 |

###  阶段三：节点识别（信号降噪过滤后）

> **核心**：在六维分析的基础上，过滤噪音信号，识别真正的交易节点。

#### 3.1 信号降噪过滤

| 过滤类型 | 屏蔽内容 | 保留内容 |
|----------|---------|---------|
| 消息噪音 | 各类传闻、自媒体吹票 | 政策文件、官方数据 |
| 情绪噪音 | 个人恐惧/贪婪 | 客观情绪数据 |
| 他人噪音 | 评论区观点、大V推荐 | 盘面语言、资金动向 |
| 历史噪音 | 过去成功路径依赖 | 当前数据、实时信号 |

#### 3.2 六类节点识别

| 节点类型 | 关键识别 |
|----------|---------|
| **① 情绪节点** | 冰点节点→试错窗口；回暖节点→加仓窗口；高潮节点→减仓窗口；退潮节点→空仓窗口 |
| **② 题材节点** | 启动→确认→加速→分歧→结束，五阶段对应不同操作 |
| **③ 量价节点** | 放量突破→新趋势启动；缩量回调→健康洗盘；地量冰点→拐点临近；放量滞涨→主力出货 |
| **④ 盘口节点** | 竞价量异动、扫板盘口、14:30后回封、封单变化、破均线 |
| **⑤ 窗口节点** | 时间窗口（10:30前涨停）、空间窗口（均线/支撑位）、情绪窗口、题材窗口 |
| **⑥ 强弱路径节点** | 加速→分歧→再加速→再分歧→结束，识别路径位置做对应操作 |
