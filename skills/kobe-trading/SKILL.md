## 一、核心世界观：大道无形，无所不及

真正的悟道不是限制于某种手法（低吸/追涨/打板/竞价/首板/1进2/龙头/反包），而是**买入即是赚钱的点**。所有手法只是工具，看懂情绪周期才是道。

**短线本质**：筹码交换的游戏，核心是人气带来的流动性溢价。逻辑和基本面只是讲故事吸引接盘资金的手段。

## 二、情绪周期四阶段（最核心框架）

```
低位震荡 → 主升 → 高位震荡 → 主跌 → （回到低位震荡）
```

### 第一阶段：低位震荡（试错期）

**特征**：接在主跌亏钱阶段之后，市场恐慌情绪未散，题材试错
**操作策略**：
- 积极试错新题材，控制仓位
- 手法：首板、1进2
- 核心逻辑：**龙头见顶杀高位阶段，是最好的布局低位新题材的时机**
- 典型案例：轴研科技2板切入吃4板；宏润建设3板隔夜顶一字吃4板

**关键判断**：
- 老龙头是否确认见顶？
- 是否有新题材出现（新闻催化）？
- 新题材的逻辑是否能发酵？

### 第二阶段：主升阶段（重仓期）

**特征**：题材共识形成，龙头持续打高度，梯队完整
**操作策略**：
- 重仓参与龙头
- 介入方式：不在乎打板还是低吸，**看好，分歧，买入**
- 核心能力：**题材预判能力**——你觉得这个题材的故事还能讲，就要敢在分歧时介入

**关键判断**：
- 题材能不能持续？故事有多大？
- 龙头是否"脖子硬"（有分歧转一致的能力）？
- 成交量是否健康？换手是否充分？
- 板块是否有跟风梯队？

**科比的经典龙头案例**：星期六、省广集团、王府井

### 第三阶段：高位震荡（防守期）

**特征**：龙头高位分歧加大，有两种可能——直接转主跌、走二波穿越
**操作策略**：
- **轻仓应对**，不建议博弈穿越（性价比低）
- 不碰中位股（中位股亏钱效应最高）
- 如果做，只做最强龙头，要么绝对低位补涨

**关键判断**：
- 亏钱效应是否在扩散？
- 龙头断板后是A杀还是横盘？
- 高标是否出现大面积跌停？

### 第四阶段：主跌阶段（空仓/防守期）

**特征**：情绪杀跌，亏钱效应扩散
**操作策略**：
- **强制空仓 / 尾盘轻仓试探**
- 不抄底、不接力、不幻想
- 第一天的杀跌很恐怖——观望
- 第二天的杀跌如果还持续——**尾盘可以找买点**
- 第三天肯定会有情绪修复——可以小仓试错

**铁律**：
- 退潮期管住手，宁可错过不可做错
- 踏空的焦虑远不如套牢的痛苦
- 看不懂行情就休息

## 三、三大核心策略：龙头 · 补涨 · 切换

### 1. 龙头

**识别标准**：
- 连板高度最高
- 涨停时间最早
- 成交额>10亿（有流动性）
- 板块跟风强（有梯队）
- 启动价格低（低价是妖股核心）
- 每次龙头都要复盘总结：成交量、换手、氛围

**操作要点**：
- 市场都认为是龙头时，买就行了
- 龙头的命硬
- 介入龙头的方法：看好→分歧→买入，不打板也可以低吸
- 持有取决于对题材深度的判断（认识过深容易亏损不走，认识过浅容易错失）

**龙头经典案例**：
- 东方通信、贵州燃气、万兴科技、君正集团
- 星期六、省广集团、王府井
- 顺控发展（14板，深交所监管松）

### 2. 补涨

**时机判断**：
- 在龙头主升过程中做（赚钱效应最好的时候）
- 在高位出现亏钱效应时反而不要做补涨（切换到防守）
- 补涨出现在"主升+高位震荡"两个阶段，前提是这轮周期没有巨大亏钱效应

**操作要点**：
- 对手速要求高，眼快手快抢好东西
- 成功率较高，但稍慢一步就买不到
- 同题材补涨：龙头打出高度→低位首板/二板做补涨
- 补涨演绎方式更暴利（凯撒旅业对王府井的补涨9个一字板）

**高低切模式**：龙头见顶期，做低位新题材的首板/二板，卡在高位分歧转低位试错的节点。

### 3. 切换

**时机判断**：
- 情绪大幅受挫之后（主跌段之后的试错期）
- 老题材逻辑失效，需要新方向

**操作要点**：
- 需要不断尝试、不断失败
- 控制仓位去感受，寻找新的聚焦点
- 切换的目标一定要与之前的核心**完全不同**（敌人的敌人就是朋友）
- 切换手法通常在**市场分歧**的时候做

**切换的经典时机**：
- 主流题材出现亏钱效应时 → 切换到注册制次新股
- 没有主流板块时 → 才做次新股、高送转等
- 主跌之后 → 必定有切换的蛛丝马迹

### 核心铁律：要么做高，要么做低，不做中位

宁可吃龙头跌停面，不碰中位涨停板。

## 五、风控与仓位管理

1. **永远分仓**：小资金也要分仓，不加杠杆
   - 做股票是长期活动，不能压在一只票上
   - 靠稳定成功率复利，不是靠梭哈运气

2. **止损纪律**：
   - 一旦不及预期，杀伐果断，立刻走人
   - 超短做隔日，不达预期就走
   - 淘汰题材反弹就走，别等
   - 绝不补仓摊平亏损

3. **回撤控制**：
   - 回撤超过15%降低仓位
   - 主跌阶段强制空仓
   - 每个月不能大起大落，每次亏损要能赚回来

4. **仓位建议**：
   - 低位试错期：1-3成
   - 主升期：5-8成（分仓龙头+补涨）
   - 高位震荡期：2-3成
   - 主跌期：0成或1成尾盘试错

## 六、题材预判能力（核心差异化能力）

科比的体系中最难复制的能力：

1. **信息渠道**：先于市场发现可发酵的舆论事件
2. **过滤能力**：大量无用信息中提取有用信号
3. **发酵判断**：判断市场是否会发酵、是否有利润空间
4. **深度把控**：
   - 认识过深→套住不走→亏损
   - 认识过浅→错失龙头
   - 合适的深度=龙头介入、持有和卖出的基础

**经典预判案例**：
- 口罩题材：江南高纤，振德医疗
- 微信电商：友阿股份
- 抖音小店：华扬联众
- 地摊经济：百大集团

## 八、结合A股当前行情的实操指南

### 第一步：判断当前情绪周期阶段

每日开盘前必做：
1. 看昨日涨停表现指数（883900）的走势
2. 看最高连板高度和晋级率
3. 看跌停数量（>10只说明亏钱效应明显）
4. 看昨日涨停票今天的溢价情况

### 第二阶段特征速查表

| 阶段 | 连板高度 | 涨停数 | 跌停数 | 亏钱效应 | 操作 |
|------|----------|--------|--------|----------|------|
| 低位震荡 | 2-4板 | 30-50 | <10 | 递减 | 首板试错 |
| 主升 | 5板+ | 60-100+ | <5 | 极小 | 重仓龙头 |
| 高位震荡 | 4-6板震荡 | 40-60 | 5-15 | 扩散中 | 轻仓防守 |
| 主跌 | 高度压缩 | <30 | >15 | 极大 | 空仓 |

### 每日操作流程

**09:15-09:25 竞价阶段**
- 看集合竞价：龙头是否超预期/不及预期
- 弱转强不要体现在竞价（竞价高开是做盘痕迹太重）
- 弱转强应该是小低开或平开后被资金买上去

**09:30-10:00 开盘确认**
- 看昨日涨停票的溢价情况
- 看龙头股的开盘走势
- 看新题材的联动效应
- 如果龙头分歧(低开但承接强) → 可以考虑介入

**10:00-11:30 盘中观察**
- 缩量板不好接（第二天资金不愿接力）
- 换手出妖股（70%以上死亡换手第二天不易弱转强）
- 反包2板是大资金喜欢的买点
- 龙头股炸板可以不走，一般票炸板要走

**13:00-15:00 尾盘决策**
- 如果当天主跌延续 → 尾盘可轻仓找买点博修复
- 观察新题材是否启动
- 准备第二天的计划

### 应对当前A股环境的具体策略

**当前市场特点（2025年）**：
- AI打首板抢筹太快，纯手打很难抢到
- 量化资金占比大，波动加剧
- 监管环境变化（深交所比上交所监管松）
- 趋势股与情绪股并存

**策略调整**：
1. 首板难打 → 多做1进2确认换手板
2. 量化波动大 → 分仓更分散，单票不超过2成
3. 监管松的交易所 → 优先做深交所的妖股
4. 逻辑+情绪结合 → 不能只看图形，要看基本面故事能讲多大
5. 趋势股也是投机 → 高位信中远海控、天齐锂业等，不止损回撤巨大
